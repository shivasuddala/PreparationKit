🔧 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: Port 8080 already in use
Solution: Kill process on 8080 or change port in application.properties

Issue: Frontend can't connect to API
Solution: Check backend is running, verify console for CORS errors

Issue: No questions showing
Solution: Wait 2-3 seconds for data seeding, then refresh browser

Issue: Database empty
Solution: Check DataSeeder.java ran on startup, restart app
