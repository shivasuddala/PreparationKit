## 🔧 CUSTOMIZE (Optional)

### Add More Questions
1. Open `src/main/java/org/example/config/DataSeeder.java`
2. Add new Question and Answer objects
3. Run: `mvn clean package` then `mvn spring-boot:run`
4. Your new questions will appear

### Change UI Styling
1. Edit `frontend/src/components/*.css` files
2. Or edit `frontend/src/index.css` for global styles
3. Changes reload automatically with `npm start`

### Modify Topics
1. Edit DataSeeder.java
2. Change topic names
3. Change subtopic names
4. Add new topics/subtopics

### Update API
1. Add new endpoints in Controllers
2. Add logic in Services
3. Add queries in Repositories
4. Test with curl or Postman
