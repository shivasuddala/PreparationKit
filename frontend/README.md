# 📚 Interview Preparation Kit

A comprehensive, advanced interview preparation web application built with React. Master Java, Spring Boot, CI/CD, and Database/SQL concepts from fresher to senior level.

🔗 **Live Demo**: [https://shiva51.github.io/PreparationKit](https://shiva51.github.io/PreparationKit)

## ✨ Features

- **4 Technology Tracks**: Java, Spring Boot, CI/CD & DevOps, Database & SQL
- **Experience Level Filtering**: Fresher, Junior, Mid-level, and Senior topics
- **Detailed Study Mode**: In-depth explanations with multiple code examples
- **Last Minute Prep Mode**: Quick bullet-point revision guides
- **Code Playground**: Built-in compiler for Java, SQL, Python, JavaScript
- **PDF Export**: Download individual topics or complete study guides
- **Responsive Design**: Works seamlessly on desktop and mobile
- **Dark Theme**: Easy on the eyes for extended study sessions

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/shiva51/PreparationKit.git
cd PreparationKit/frontend

# Install dependencies
npm install

# Start development server
npm start
```

The app will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```
